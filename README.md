## 声明
本项目使用GPLv3许可

本项目不带任何代理功能

仅供GUI学习交流使用

请勿用于违法用途

## 当前特性
- 支持启动/重启/停止
- 支持更新核心文件[如果有http入口则走http入口]
- 支持导入config.json文件
- 显示socks和http端口
- 间隔5s测试一次通过代理打开谷歌所需的时间
- 右击最小化按钮可最小化到系统托盘

## 使用方式
