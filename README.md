# V2py

## 特性
- 支持启动/重启/停止
- 支持更新核心文件
- 支持从别的地方导入config.json文件配置
- 显示socks端口
- 间隔5s测试一次通过代理打开谷歌所需的时间
- 右击最小化按钮可最小化到系统托盘

## 使用方式
1. 将文件放在同一文件夹下并在建立名为"V2ray-core"的文件夹
2. 将核心文件解压到该文件夹下
3. 导入/编辑配置文件
4. 启动主程序V2py.py或V2py.exe
